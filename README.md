## 소개
유료 Open AI 달리와 스테이블 디퓨전의 API를 연동하여 이미지, 3D모델, 영상을 생성할 수 있는 gen- ai hub Test service로 
NextJs로 생성된 데이타를 public 폴더에서 관리하도록 구현하였습니다.  

## 기능
- 생성형 AI
